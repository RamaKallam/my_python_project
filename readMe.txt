# Python & Data Science Learning Repository

## Overview
This repository contains structured learning notebooks covering Python programming fundamentals, data analysis, data visualization, and machine learning basics. 
It demonstrates hands-on practice with commonly used Python libraries in data science and AI workflows.

The goal of this project is to build a strong foundation in Python and develop practical skills required for Data Analyst / Data Scientist / AI Engineer roles.

---

## Topics Covered

### Python Fundamentals
- Variables, data types, and type conversion  
- Input/output operations  
- Conditional statements and loops  
- Error types and debugging basics  

### NumPy
- Arrays (1D, 2D, 3D)  
- Indexing and slicing  
- Mathematical operations  
- Linear algebra basics  
- Random number generation  

### Pandas
- Series and DataFrames  
- Data selection and filtering  
- Working with structured datasets  
- Loading and exploring data  

### Data Visualization

### Machine Learning (Scikit-learn)

  - Decision Tree Classifier  
- Model evaluation:
  - Confusion matrix  
  - Accuracy, precision, recall  
- Iris dataset classification

## Machine Learning Project – Iris Classification

This project demonstrates an end-to-end machine learning workflow including data preprocessing, model training, evaluation, and prediction using Python and Scikit-learn.

### Models Used:
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier

### Workflow:
1. Load dataset  
2. Perform exploratory data analysis (EDA)  
3. Preprocess data (scaling & splitting)  
4. Train models  
5. Evaluate performance  

### Outcome:
Both models achieved high accuracy on the dataset, demonstrating understanding of classification pipelines.
